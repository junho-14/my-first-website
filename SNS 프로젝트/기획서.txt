=====================================
모바일 서비스 UI 기획서
=====================================

1. 서비스 기본 정보
-------------------------------------
- 서비스 이름: T.M.I(Time Match Integration)
- 서비스 주제: 친구, 동료 등 여러 사용자의 일정을 공유하고 공통으로 가능한 시간을
자동으로 찾아 약속을 쉽게 생성할 수 있는 일정 조율 서비스
- 핵심 가치: 그룹 구성원의 일정 중 약속이 가능한 공통 시간을 자동으로 탐색하여
일정을 확인하고 조율하는 과정에서 발생하는 불편을 줄인다.
- 해결하고자 하는 문제:
여러 사람이 약속을 잡을 때 각자의 가능한 시간을 메신저로 반복해서
확인해야 하며, 참여 인원이 많아질수록 일정 조율에 많은 시간이 소요된다.
T.M.I는 개인 일정과 그룹 일정을 연동하고 공통 가능 시간을 자동으로
계산하여 이러한 문제를 해결한다.
- 타겟 사용자:
20~40대 대학생 및 직장인 중 친구, 동료, 프로젝트 팀원 등과
약속이나 회의 일정을 자주 조율하는 사용자
- 주요 이용 흐름:
로그인
→ 개인 일정 등록
→ 그룹 생성 또는 참여
→ 그룹원 일정 확인
→ 공통 가능 시간 탐색
→ 약속 생성
→ 그룹원 캘린더 반영

2. 디자인 컨셉
-------------------------------------
메인 색상:
    - 파스텔 블루(#7CC6FE 계열)
    - 주요 버튼, 선택된 메뉴, 공통 가능 시간 강조에 사용
보조 색상:
    - 파스텔 핑크: 개인 일정
    - 파스텔 그린: 그룹 약속
    - 파스텔 옐로우: 중요 일정
    - 라이트 그레이: 배경, 구분선 및 비공개 일정의 시간 블록
전체 분위기:
    밝고 깔끔한 미니멀 디자인을 기반으로 파스텔톤 색상을 활용하여
    친근하고 편안한 분위기를 제공한다. 흰색 배경을 중심으로 일정의
    종류에 따라 색상을 구분하여 가독성을 높이고, 처음 사용하는 사람도
    직관적으로 기능을 이해할 수 있는 UI를 목표로 한다.
UI 원칙:
    - 한 화면에 지나치게 많은 정보를 표시하지 않는다.
    - 주요 기능은 2~3번 이내의 터치로 실행할 수 있도록 구성한다.
    - 색상뿐만 아니라 텍스트와 형태를 함께 사용하여 상태를 구분한다.
    - 공통 가능 시간과 선택된 시간은 명확하게 강조한다.
    - 일정 제목이 길 경우 말줄임표로 처리한다.
참고 서비스:
    - Google Calendar
    - TimeTree
    - Apple Calendar
    - Notion Calendar

3. 필요한 페이지 목록
-------------------------------------
1. 로그인 / 회원가입 페이지
    - 기능:
      · 이메일 회원가입 및 로그인
      · Google, Kakao, Apple 계정 로그인
      · 비밀번호 재설정
      · 이용약관 및 개인정보 처리방침 동의

    - 특징:
      · 간편 회원가입 및 로그인 지원
      · 최초 로그인 시 이름과 프로필 이미지 설정
      · 사용자 인증을 통한 안전한 일정 관리

2. 개인 캘린더 페이지(메인 화면)
    - 기능:
      · 월간 및 주간 캘린더 전환
      · 날짜별 일정 조회
      · 일정 등록 화면으로 이동
      · 오늘 날짜로 이동
      · 일정 색상 및 종류 확인

    - 특징:
      · 사용자의 기본 일정 관리 공간
      · 개인 일정과 확정된 그룹 약속을 함께 표시
      · 일정의 종류에 따라 색상을 구분
      · 비공개 여부는 일정 상세 화면에서 확인

3. 일정 등록 / 수정 페이지
    - 기능:
      · 일정 제목 입력
      · 시작 날짜와 시간 설정
      · 종료 날짜와 시간 설정
      · 종일 일정 설정
      · 반복 일정 설정
      · 장소 및 메모 입력
      · 일정 색상 설정
      · 공개 범위 설정
      · 일정 수정 및 삭제

    - 공개 범위:
      · 공개: 그룹원에게 일정 제목과 시간 표시
      · 비공개: 그룹원에게 제목과 내용은 숨기고 시간 사용 여부만 반영

    - 특징:
      · 일정 등록과 수정에 동일한 UI 사용
      · 필수 입력값이 없을 경우 안내 문구 표시
      · 종료 시간이 시작 시간보다 빠르게 설정되지 않도록 제한

4. 그룹 목록 페이지
    - 기능:
      · 참여 중인 그룹 목록 조회
      · 새로운 그룹 생성
      · 초대 코드 또는 초대 링크를 통한 그룹 참여
      · 즐겨찾기 그룹 설정

    - 특징:
      · 여러 그룹을 한 화면에서 관리
      · 그룹별 이름, 대표 이미지, 인원수 표시
      · 새로운 초대가 있는 그룹을 별도로 표시

5. 그룹 생성 / 관리 페이지
    - 기능:
      · 그룹 이름 및 대표 이미지 설정
      · 그룹원 초대
      · 초대 코드 또는 링크 생성
      · 그룹원 목록 조회
      · 그룹 탈퇴
      · 그룹 삭제

    - 관리자 기능:
      · 그룹 정보 수정
      · 그룹원 내보내기
      · 그룹 삭제

    - 특징:
      · 그룹 생성자는 기본적으로 관리자로 지정
      · 일반 그룹원과 관리자의 권한을 구분

6. 그룹 캘린더 페이지
    - 기능:
      · 그룹 약속 조회
      · 그룹원별 일정 가능 여부 확인
      · 날짜별 일정 확인
      · 공통 시간 찾기 페이지로 이동

    - 특징:
      · 공개 일정은 사용자가 허용한 정보만 표시
      · 비공개 일정은 제목과 내용을 표시하지 않음
      · 비공개 일정이 있는 시간은 사용 중인 시간으로만 표시
      · 그룹원 개인 일정의 세부 내용보다 약속 가능 여부를 중심으로 구성

7. 공통 시간 찾기 페이지
    - 기능:
      · 탐색할 날짜 또는 기간 설정
      · 희망 시간대 설정
      · 약속 예상 소요 시간 설정
      · 참여할 그룹원 선택
      · 공통 가능 시간 자동 계산
      · 추천 결과 정렬 및 선택

    - 입력 조건 예시:
      · 탐색 기간: 7월 20일~7월 27일
      · 희망 시간: 오전 9시~오후 10시
      · 약속 소요 시간: 1시간, 2시간 등
      · 참여 인원: 전체 그룹원 또는 일부 그룹원

    - 특징:
      · 개인 일정의 공개 여부와 관계없이 사용 중인 시간을 계산에 반영
      · 모든 구성원이 가능한 시간을 우선 추천
      · 공통 시간이 없을 경우 가장 많은 인원이 가능한 시간을 보조적으로 표시
      · 누가 어떤 이유로 불가능한지는 공개하지 않음

8. 약속 생성 페이지
    - 기능:
      · 추천 시간 선택
      · 약속 제목 입력
      · 시작 및 종료 시간 확인
      · 장소 및 메모 입력
      · 참여자 선택
      · 약속 생성

    - 특징:
      · 선택한 공통 시간을 기본값으로 자동 입력
      · 생성된 약속을 참여자 캘린더에 자동 반영
      · 약속 생성 완료 후 그룹 캘린더로 이동

9. 약속 상세 페이지
    - 기능:
      · 약속 제목, 시간, 장소 및 참여자 확인
      · 약속 수정 및 취소
      · 참여 여부 선택
      · 변경된 약속 내용 확인

    - 특징:
      · 약속을 수정하거나 취소할 경우 참여자에게 알림 제공
      · 약속 생성자 또는 관리자에게 수정 권한 부여
      · 일반 참여자는 참석, 미정, 불참 중 하나를 선택

10. 알림 페이지
    - 기능:
      · 그룹 초대 확인 및 수락
      · 약속 생성, 수정 및 취소 알림 확인
      · 참여 여부 요청 확인
      · 읽음 및 읽지 않음 상태 관리

    - 특징:
      · 최근 알림을 시간순으로 표시
      · 알림을 선택하면 관련 그룹 또는 약속 화면으로 이동

11. 마이페이지
    - 기능:
      · 이름 및 프로필 이미지 수정
      · 알림 설정
      · 기본 캘린더 보기 설정
      · 개인정보 및 계정 관리
      · 로그아웃 및 회원 탈퇴

    - 특징:
      · 사용자 정보와 서비스 환경설정을 한곳에서 관리

4. 핵심 기능 정의
-------------------------------------
필수 기능

1. 회원가입 및 로그인
    - 이메일 및 SNS 계정을 통한 회원가입과 로그인
    - 로그인 상태 유지 및 로그아웃
    - 사용자별 일정 데이터 분리

2. 개인 일정 관리
    - 일정 등록, 조회, 수정 및 삭제
    - 날짜와 시간, 종일 일정 설정
    - 일정 색상 및 중요도 설정
    - 월간 및 주간 캘린더 제공

3. 일정 공개 범위 설정
    - 일정을 공개 또는 비공개로 설정
    - 공개 일정은 허용된 일정 정보를 그룹원에게 표시
    - 비공개 일정은 제목과 내용을 숨기고 사용 중인 시간으로만 처리
    - 비공개 일정도 공통 시간 계산에는 반영

4. 그룹 생성 및 관리
    - 그룹 생성 및 참여
    - 초대 링크 또는 초대 코드를 통한 그룹원 초대
    - 그룹원 목록과 권한 관리
    - 그룹 탈퇴 및 삭제

5. 그룹 캘린더
    - 그룹 약속 조회
    - 그룹원의 시간 사용 여부 확인
    - 개인 일정의 공개 범위에 따른 정보 표시

6. 공통 시간 자동 탐색
    - 선택한 기간, 시간대, 소요 시간과 참여자를 기준으로 계산
    - 모든 참여자가 가능한 시간을 우선 추천
    - 공통 시간이 없을 경우 대체 가능한 시간 제공

7. 약속 생성 및 공유
    - 추천 시간을 선택하여 그룹 약속 생성
    - 약속 제목, 장소, 메모 및 참여자 설정
    - 생성된 약속을 참여자 캘린더에 자동 반영

8. 약속 변경 및 참여 상태 관리
    - 약속 수정 및 취소
    - 참석, 미정, 불참 상태 선택
    - 약속 변경 내용을 참여자에게 전달


부가 기능

1. 푸시 알림
    - 그룹 초대, 약속 생성, 변경, 취소 및 시작 전 알림

2. 일정 검색
    - 일정 제목, 날짜, 장소를 기준으로 검색

3. 약속 장소 및 위치 공유
    - 장소 주소와 지도 링크 공유

4. 즐겨찾기 그룹
    - 자주 사용하는 그룹을 목록 상단에 고정

5. 일정 가져오기 및 내보내기
    - 외부 캘린더 일정 가져오기
    - 일정 파일 또는 링크를 통한 내보내기

6. AI 기반 최적 시간 추천
    - 사용자의 선호 시간대와 기존 약속 패턴을 분석하여 시간 추천

7. 약속 장소 추천
    - 참여자 위치 또는 선호 지역을 기준으로 중간 장소 추천

8. 일정 통계
    - 일정 유형, 약속 빈도, 자주 만나는 그룹 등의 통계 제공

5. 모바일 UI 특징
-------------------------------------
네비게이션:
    - 하단 탭 네비게이션 사용
    - 구성:
      [캘린더] [그룹] [+] [알림] [마이]

    - 캘린더:
      개인 일정 및 확정된 그룹 약속 확인

    - 그룹:
      그룹 목록 및 그룹 캘린더 확인

    - + 버튼:
      개인 일정 등록 또는 그룹 약속 생성을 선택하는 메뉴 표시

    - 알림:
      그룹 초대 및 약속 관련 알림 확인

    - 마이:
      프로필과 환경설정 관리

주요 버튼 위치:
    - + 버튼(FAB): 하단 네비게이션 중앙
    - 오늘 버튼: 캘린더 화면 상단
    - 월간/주간 전환 버튼: 캘린더 화면 상단
    - 공통 시간 찾기 버튼: 그룹 캘린더 하단 또는 상단의 주요 영역
    - 약속 생성 버튼: 공통 시간 결과 화면 하단 고정

터치 영역:
    - 주요 터치 요소의 최소 크기 44 × 44px
    - 버튼 사이 간격 8px 이상
    - 자주 사용하는 기능은 엄지손가락이 닿기 쉬운 화면 하단에 배치

화면 방향:
    - 스마트폰에서는 세로 모드를 기본으로 지원
    - 태블릿에서는 가로 모드를 추가로 고려
    - MVP에서는 세로 모드를 우선 개발

UI 상태:
    - 로딩 상태: 일정 또는 추천 시간을 계산 중임을 표시
    - 빈 화면: 일정이나 그룹이 없을 때 생성 방법 안내
    - 오류 상태: 데이터 불러오기 실패와 재시도 버튼 제공
    - 완료 상태: 일정 및 약속 생성 완료 메시지 제공
    - 비활성 상태: 필수 정보를 입력하지 않으면 완료 버튼 비활성화

UI 특징:
    - 파스텔톤을 기반으로 일정 종류를 직관적으로 구분
    - 월간 캘린더에서는 일정 개수를 간단하게 표시
    - 주간 캘린더에서는 일정 시간과 길이를 블록 형태로 표시
    - 공통 가능 시간은 파스텔 블루 배경 또는 테두리로 강조
    - 비공개 일정은 회색 시간 블록으로만 표시
    - 중요한 동작 전에는 확인 창을 제공
      예: 일정 삭제, 그룹 탈퇴, 약속 취소

6. 개발 우선순위
-------------------------------------
1차 개발(MVP)
목표:
개인 일정을 등록하고 그룹을 만든 뒤 공통 시간을 찾아 약속을 생성하는
서비스의 핵심 흐름을 완성한다.

개발 기능:
    - 이메일 회원가입 및 로그인
    - 기본 프로필 설정
    - 개인 캘린더 월간 화면
    - 개인 일정 등록, 조회, 수정 및 삭제
    - 일정 공개 및 비공개 설정
    - 그룹 생성 및 참여
    - 초대 코드 또는 링크를 통한 그룹원 초대
    - 그룹 캘린더
    - 비공개 일정의 시간 블록 처리
    - 기간, 시간대 및 소요 시간을 이용한 공통 시간 계산
    - 약속 생성 및 참여자 캘린더 반영
    - 기본적인 모바일 세로 UI

MVP 완료 기준:
    사용자가 회원가입 후 개인 일정을 등록하고, 그룹에 친구를 초대하여
    공통 가능 시간을 찾은 뒤 그룹 약속을 생성할 수 있어야 한다.


2차 개발
목표:
일정 조율과 그룹 사용의 편의성을 높인다.

개발 기능:
    - Google, Kakao, Apple 소셜 로그인 확대
    - 월간 및 주간 캘린더 전환
    - 반복 일정
    - 약속 수정 및 취소
    - 참석, 미정, 불참 상태 관리
    - 푸시 알림
    - 일정 검색
    - 그룹 대표 이미지
    - 즐겨찾기 그룹
    - 관리자 및 일반 멤버 권한 구분
    - 공통 시간이 없을 때 대체 시간 제공


3차 개발
목표:
외부 서비스 연동과 개인화 기능을 통해 서비스 경쟁력을 강화한다.

개발 기능:
    - AI 기반 최적 약속 시간 추천
    - 참여자 위치 기반 장소 추천
    - Google Calendar 등 외부 캘린더 연동
    - 일정 가져오기 및 내보내기
    - 일정 및 그룹 활동 통계
    - 태블릿 및 가로 모드 최적화
