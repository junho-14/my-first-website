== 나만의 커뮤니티 기획서 ==

커뮤니티 주제

-   게임 커뮤니티
-   다양한 게임 정보를 공유하고 공략, 후기, 자유로운 소통을 할 수 있는
    커뮤니티
-   PC 게임, 모바일 게임, 콘솔 게임을 모두 지원

디자인 스타일

-   사이트 이름: GameHub
-   색상 테마
    -   메인: 다크 블루 (#1E3A8A)
    -   포인트: 하늘색 (#38BDF8)
    -   배경: 밝은 회색 (#F8FAFC)
-   분위기
    -   게임 감성이 느껴지는 깔끔한 UI
    -   직관적인 디자인
    -   카드형 레이아웃
    -   반응형 웹(PC / 태블릿 / 모바일)

특별한 기능 아이디어

기본 기능

-   회원가입
-   로그인 / 로그아웃
-   게시글 작성, 수정, 삭제
-   댓글 작성, 수정, 삭제
-   게시글 검색
-   게시글 좋아요 / 싫어요

추가 기능

-   인기 게시글 자동 선정
-   게임별 게시판
-   조회수 표시
-   댓글 좋아요
-   프로필 이미지

-   최근 작성 게시글 / 댓글
-   이미지 업로드
-   다크모드
