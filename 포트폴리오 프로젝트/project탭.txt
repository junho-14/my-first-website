썸네일 카드 구성
이미지:
- 타입: [O] 스크린샷 [O] 모바일 목업 [ ] 커스텀 디자인
- 비율: [ ] 16:9 [ ] 4:3 [O] 1:1
- 크기: 가로 300px, 세로 300px

정보 표시:
- 프로젝트 이름 (제목)
- 한 줄 설명: _______________
- 기술 스택: [ ] 뱃지형 [O] 텍스트형 [ ] 아이콘형
- 기타 정보: (개인/팀)

링크 및 버튼:
- [O] Live Demo 버튼
- [O] GitHub 버튼

인터랙션 효과
- 호버 효과: 카드 테두리 네온효과
- 클릭 피드백: 버튼 애니메이션
- 모바일 터치: 터치 피드백
